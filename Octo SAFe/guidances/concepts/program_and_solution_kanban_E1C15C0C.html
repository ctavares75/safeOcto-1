Concept: Program and Solution Kanban
The Program and Solution systems are a method to visualize and manage the flow of Features and Capabilities from ideation to analysis, implementation, and release through the Continuous Delivery Pipeline. The program Kanban facilitates the flow of features through the continuous delivery pipeline.

Main Description

The Kanban systems help Agile Release Trains (ARTs) and Solution Trains match demand to capacity based on Work in Process (WIP) limits, and visualizing bottlenecks in each process state helps identify opportunities for relentless improvement. The Kanban system also includes policies governing the entry and exit of work items in each state.

These systems provide many benefits. They:

-Increase visibility into existing and upcoming work, and better understand the flow of work

-Ensure continuous refinement of new value definition and acceptance criteria

-Foster collaboration across disciplines, functions, and levels

-Support economic decision-making by setting the policies for the pull-based mechanism

-Establish connections between the ARTs, Solution Train, and Portfolio
