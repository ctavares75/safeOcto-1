Welcome
Scaled Agile Framework (SAFe) is a software development framework that consists of a bas of patterns intended for enterprise-scale Lean-Agile development. SAFe is scalable and modular, such that an organization may apply its model in any way needed.
Main Description

SAFe consists of four core values that guide behavior and action. These help direct workers' focus and keep business on track. They are:


1. Alignement

2. Built-In Quality

3. Transparency

4. Program Execution
